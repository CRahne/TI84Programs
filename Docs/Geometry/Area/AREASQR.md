# Area of a Sqaure
```
:ClrHome
:Disp "AREA OF SQUARE"
:Input "DIS OF ONE SIDE ",X
:Disp (X*X)
```

#### Program Desc

This will ask for the length of the square. It will then display the area in units^2

#### Equation

area = l^2. This is similar to a reactangle, but the sides are the same length so it works by just squaring one side. [Wikipedia](https://en.wikipedia.org/wiki/Square)
